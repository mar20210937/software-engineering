## 2. Overview
The Parking Management and Guidance System is a smart system designed to monitor parking availability, guide users to available parking spots, and manage parking operations efficiently.

The system allows users to:
- View available parking spaces  
- Reserve parking spots  
- Receive navigation guidance  
- Pay parking fees  

Administrators can:
- Manage parking spaces  
- Set pricing policies  
- Monitor occupancy  
- Generate reports  

### Tools Used
- Visual Studio Code  
- PlantUML  
- Draw.io
